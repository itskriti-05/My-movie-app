# 🎬 CineVault

A sleek and modern movie discovery app built with React and the TMDB API. CineVault lets you explore what's trending, search for any film, and build your own personal movie collection with favorites and a watchlist — all saved locally so nothing gets lost on refresh.

## Features
- 🔥 Browse trending movies updated weekly
- 🔍 Search any movie from TMDB's massive database
- ★ Save favorites to revisit anytime
- 📋 Build a watchlist for movies you plan to watch
- 🎭 View cast, genres, ratings, runtime & overview
- 💾 Favorites and watchlist persist via localStorage

## Tech Stack
- React + Vite
- TMDB API
- LocalStorage for persistence
